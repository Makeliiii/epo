# EPO
This repository stores the codebase for [Oulun Etel√§- ja Keskipohjalainen Osakunta (EPO ry)](http://www.student.oulu.fi/~epo/) website. EPO ry is a student organization targeted for students originating from the Southern and Central Ostrobothnia regions. EPO arranges all kinds of activities for its' members in the form of different events and activities.

## Install
``` shell
# npm
npm install

# yarn
yarn add
```

## Usage
1. Install [Node.js](https://nodejs.org/)
2. Install dependencies as described [here](https://github.com/Makeliiii/epo#install)
3. Run ```npm start``` to start development server
