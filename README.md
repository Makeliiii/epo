## Install
``` shell
# npm
npm install

# yarn
yarn add
```

## Usage
1. Install [Node.js](https://nodejs.org/)
2. Install dependencies as described [here](https://github.com/Makeliiii/epo#install)
3. Run ```npm start``` to start development server
